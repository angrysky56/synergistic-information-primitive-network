# Tech Stack

## Core Technologies
- **Language**: Python 3.13
- **Deep Learning**: PyTorch 2.10.0
- **Scientific Computing**: NumPy, SciPy

## Development Tools
- **Package Manager**: `uv`
- **Linting**: Ruff (strict configuration)
- **Type Checking**: Mypy (strict mode)
- **Testing**: Pytest

## Infrastructure
- **Information Theory**: Custom estimators for PID (Partial Information Decomposition), AIS (Active Information Storage), and TE (Transfer Entropy).
- **Network Architecture**: Custom "Synergistic Information Primitive Network" (SIPNet) built on PyTorch.
