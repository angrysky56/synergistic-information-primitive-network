# SIP-Net: Architecture Overview

Synergistic Information Primitive Network (SIP-Net) is a hierarchical neural architecture designed for high-fidelity information processing through the lens of partial information decomposition (PID).

## Core Principles

### 1. Information Primitives
Information is processed as a set of synergistic, redundant, and unique primitives. Unlike standard neural networks that aggregate features linearly or non-linearly, SIP-Net explicitly optimizes for **synergy**—information that only exists in the interaction of multiple inputs.

### 2. Hierarchical Synergy
Phase 6 introduces a multi-layer hierarchical structure. Each layer consists of:
- **SIPLayer**: A collection of nodes that compute information-theoretic metrics over their inputs.
- **Residual Connections**: Stable identity paths that allow gradients to flow through deep stacks (3-12 layers) while preserving sensory signals.
- **Inter-Layer Synergy**: A specialized loss component that encourages layer $L+1$ to capture information that is synergistic with respect to layer $L$, preventing redundant representation.

## Component Breakdown

### Sensory Encoders
SIP-Net supports two primary sensory modes:
- **Direct Input**: Standard vector embeddings or raw features.
- **Transformer Encoders**: Native integration with Hugging Face `transformers` (BERT, RoBERTa, etc.). This allows the network to process raw text and align subtokenized embeddings with word-level annotations.

### IPP (Information Processing Primitives) Graph
The heart of the network is a dynamic graph where edges represent information transfer ($TE$) and nodes represent storage ($AIS$). The network evolves its internal topology based on the information flow requirements of the task.

### Application Heads
The network is task-agnostic but currently optimized for:
- **Coreference Ranking**: A mention-ranking head that uses SIP-Net's final representation to calculate scores across entity clusters.

## Information Metrics

SIP-Net uses Matrix-Rényi estimators to calculate:
- **Synergy ($Syn$)**: Non-redundant, interaction-based information.
- **Transfer Entropy ($TE$)**: Directed information flow between nodes.
- **Active Information Storage ($AIS$)**: Predictive information held within a node's state.

## Scaling to Real-World Data
With the integration of OntoNotes 5.0, SIP-Net scales from synthetic toy tasks to industrial-grade coreference resolution. This is achieved through:
- **Large-Batch Estimators**: Optimized entropy calculations for high-dimensional hidden states.
- **Subtoken-Word Alignment**: Precise mapping between transformer outputs and linguistic mentions.
