# SIP-Net: Testing & Quality Assurance

## Testing Suite

We use `pytest` for unit and integration testing.

```bash
# Run all tests
export PYTHONPATH=$PYTHONPATH:$(pwd)/src
uv run pytest
```

## Key Test Categories

### 1. Unit Tests
- `tests/unit/domain/`: Mathematical correctness of estimators and graph logic.
- `tests/unit/infrastructure/`: Data loading and metric accuracy.

### 2. Integration Tests
- `tests/integration/`: End-to-end training loops and multi-layer convergence.

### 3. Stability & Scaling
- `scripts/test_multi_layer_scaling.py`: Checks VRAM and latency for deep architectures.
- `scripts/test_batch_scaling.py`: Ensures Matrix-Rényi estimators scale with batch size.

## Type Checking

We enforce strict typing using `mypy`.
```bash
uv run mypy src/sipnet
```

## Continuous Integration

Before submitting a PR, ensure:
1. `pytest` passes with 0 failures.
2. `mypy` reports "Success: no issues found".
3. `CognitiveTrainer` successfully completes a phase-1 synthetic XOR run.
