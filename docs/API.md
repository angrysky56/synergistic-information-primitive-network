# SIP-Net: API Reference

## Data Infrastructure

### `sipnet.infrastructure.data.ontonotes_loader`
Loader for the CoNLL-2012 (OntoNotes 5.0) dataset.

- **`CoNLL2012Dataset`**:
  - `path`: Path to the `.conll` file.
  - `tokenizer_name`: HF tokenizer name (e.g., `bert-base-uncased`).
  - **Returns**: A dataset yielding `(input_ids, attention_mask, word_map, clusters)`.
- **`ontonotes_collate_fn`**:
  - Batches variable-length documents and clusters into a padded format suitable for training.

## Network Domain

### `sipnet.domain.network.graph.SIPNet`
The main network entry point.

- **Parameters**:
  - `input_dim`: Input dimension (ignored if `hf_model_name` is set).
  - `hidden_dim`: Internal state dimension.
  - `num_layers`: Depth of the hierarchical stack.
  - `hf_model_name`: Optional Hugging Face model name for sensory encoding.
  - `hf_freeze`: Whether to freeze the transformer encoder.

### `sipnet.application.execution.coref_head`
Specialized head for coreference tasks.

- **`CorefMentionRankingHead`**:
  - Takes `hidden_dim` as input.
  - Computes antecedent scores for all mention pairs in a document.

## Training & Optimization

### `sipnet.application.training.loss_function.CompositeLoss`
Multi-objective loss function.

- **Components**:
  - `task_loss`: Standard supervised loss (BCE, CrossEntropy).
  - `synergy_weight`: Weight for information-theoretic synergy.
  - `inter_layer_synergy`: Encourages novel information transfer between layers.

### `sipnet.application.training.trainer.CognitiveTrainer`
Orchestrates phase-based training.

- **`set_phase(phase_idx)`**:
  - Configures active layers and parameter freezing.
  - Implements "Cognitive Phasing" where lower layers are stabilized before higher layers are engaged.
